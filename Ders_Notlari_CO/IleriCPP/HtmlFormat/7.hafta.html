Space Ship Operatr

Bu operatr ounlukla dorudan kullanmyoruz. Kullandmz yer kendi trlerimiz iin bu operatr overload ettiimiz yerler. Artk operatrler arasnda bir ayrm yaplyor ve bu ayrm:

  1. == ve <=> operatr
  2. !=, >, <=, >=, > operatrleri
  • Primary operatrler reversable (derleyici uygun fonksiyon bulunmamas durumunda operandlar yer deitirebiliyor).
  • Secondary operatrler ise rewritable ( eer x!=y yok ise bunu !(x==y) olarak deitirebilir.)

Bu da kendi snflarmz iin != deili yazmamza gerek yok == yazmamz yeterli olacak. Ayns dier operatrleri yazmak yerine <=> operatrn yazmamz yeterli olacak.

  • eler bildirim srasna gre karlatrlr.

  • == fonksiyonu iin aadaki rtl dnmler yaplyor.

class Myclass {
public:
    Myclass(int x = 0) : mx{x} {}
    bool operator==(const Myclass &other) const
    {
        return mx == other.mx;
    }
private:
    int mx;
};

int main()
{
    Myclass x{10}, y{20};
    auto b1 = x == y; // x.operator==(y)
    auto b2 = x != y; // !(x.operator==(y))
    auto b3 = x == 5;
    auto b4 = 5 == x; //C++20 ncesi bu legal deildi, derleyici 5 == x'iin x.operator==(5) arsna dntryor.
}

  • C++11 standart ile dile eklendi.
